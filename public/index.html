About TikCat API

TikCat API provides a complete event-ticketing backend solution for developers and organizations. Create events, manage venues, sell tickets, process orders, authenticate users, and validate tickets using QR codes.

Core Features

Event Management

Create and manage public or private events.

Venue Management

Manage event locations and capacities.

Ticket Generation

Automatically generate unique tickets.

QR Code Validation

Validate tickets securely at entry points.

Order Processing

Handle purchases and ticket reservations.

OAuth Authentication

Google and GitHub login integration.

Dashboard Analytics

Track revenue, orders, events and users.

Swagger Documentation

Interactive API testing and exploration.

Who Uses TikCat API?

Event Organizers

Create, manage and monitor events.

Venue Managers

Developers

Attendees
